モジュール一覧

Favorites
  • B キーで現在ホバー中のアイテムを登録/解除。
  • クリエイティブの「お気に入り」タブに即反映。
  • 永続データは .minecraft/config/building-support-favorites.json。
History
  • 設置/右クリック/ピックブロックを監視して最新64件を保持。
  • Config で「ワールド別」or「全ワールド共有」を切替。
  • .minecraft/itemgroups/history/<world>.json へ保存。
Building Tabs
  • 木材/石材/銅/光源/ネザー/エンドの6カテゴリ。
  • 専用クラスで定義し、Vanilla + Fabric API を幅広く網羅。
  • Mod Menu → 「インベントリ制御」で概要確認。

Commands コマンド & サーバー管理

コマンドプリセットと村ワープ機能の詳細、設定フローを一箇所にまとめました。

← トップへ戻る

ユーティリティ

/busp preset
  • add / remove / list でプリセット管理。
  • 設定ファイル: .minecraft/config/building-support-commands.json。
  • ショートカットでよく使うコマンドを呼び出し。
/village suite
  • /village → 2番目に近い村へ即テレポート。
  • /village new <distance> → 未訪問の村探索。
  • Config「スポーン」で対象バイオームと自動スポーン制御。
